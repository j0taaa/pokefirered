# ts-game (TypeScript Browser Port Target)

This directory is the destination for the browser-playable TypeScript port of Pokémon FireRed.

## Initial scope

- Provide a standalone TypeScript web app scaffold
- Host game runtime systems that are not tied to GBA build tooling
- Grow in small vertical slices (rendering, input, movement, maps, battle, UI)

## Planned top-level layout

- `src/` — TypeScript source
- `public/` — static web assets

(Implementation to follow in later commits.)
